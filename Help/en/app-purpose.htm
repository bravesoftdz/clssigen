Client SSI Generator

Purpose of the program

Client SSI Generator is a complementary tool for web developers and designers and is intended for imitating processing the SSI content by a server, but on the client-side. It can be used in cases SSI service is unavailable at the HTTP server, and higher-level services (CGI, ASP or PHP) are disabled or unavailable as well.

The program requires ca 2 MB free space on the hard disk and 8 MB (or more, for Windows to effectively cache files) of physical memory.
Operating system: Windows 95/98/ME/NT/2000/XP.
